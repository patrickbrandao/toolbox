

ToolBox

	Binario unificado com todos os pequenos comandos utilizados em shell-scripts.
	Comandos que aceleram tarefas e executam funcoes com o minimo de uso de CPU e memoria, visam subistituir muitas
	linhas de shell-script e facilitar o uso do sistema linux


Arquivos:
	_commands.txt	-> arquivo com declaracao de comandos e apelidos a serem incluidos no projeto
	_make.sh 		-> cria o binario final, unificando todos os sub-programas e compilando

	_main.c 		-> codigo principal, sera usado como base para gerar o codigo unificado

	libtoolbox.h 	-> biblioteca de definicoes centralizadas

Projetos:
	common/ 		-> Programas comuns sem classificacao
	shellcolor/		-> Manipulacao de terminal, cores, echo colorido, saida formatada
	strings/		-> funcoes de strings
